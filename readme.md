# Student Performance Predictor

A machine learning project that predicts a student's final grade (G3) based on their background, study habits, and academic records.

##  Features
- Data preprocessing and cleaning
- Linear Regression model training
- Interactive Streamlit web app for predictions

##  Tech Stack
- Python
- Pandas, Scikit-learn
- Streamlit
- Joblib
